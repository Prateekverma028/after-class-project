Inline CSS

CSS can be directly added within HTML code using Inline CSS.

In Inline CSS, the style attribute is directly added to the opening tag of the relevant element. Any CSS property can be added to the style attribute.

Internal CSS

In Internal CSS STYLE element is added inside HEAD section of the HTML page.

You need to write the CSS code inside the opening and closing STYLE tag.

External CSS

In External CSS, a CSS file is linked to the HTML file by using a LINK element inside HEAD section.

This is done by providing a reference to external style sheet inside the LINK element.Using external CSS is considered the best approach

SELECTORS

Selectors help you can select the content you want to style. Selectors are part of the CSS ruleset.

They select HTML elements according to there Id,Class,Type,Attribute,etc.

The diffrent type of selectors in CSS are:-
  • CSS Element Selector
  • CSS Id Selector
  • CSS Universal Selector
  • CSS Group Selector

Element Selector

The element selector selects the HTML elements based on the element name. For eg: h1,p.,div,h2,etc.

Id Selector

In this selector, the id attribute of an HTML element is used to select a specific element. It can only select one unique element since an element has a unique id within a page.

To select an element with a specific id, a hash (#) character, followed by the id of the element is used.

Class Selector

Class selector is used to select HTML elements with a specific class attribute.

A period (.) character, followed by the class name, is used to select elements with a specific class.

Universal Selector

The universal selector(*) selects all HTML elements on the page

Group Selector

Group selector selects all the elements with the same style. It is used to minimise the code. Each selector is separated by using a comma(,).
